# Meta Front-End Developer Professional Certificate Projects

This repository contains the hands-on projects I built while completing the **Meta Front-End Developer Professional Certificate** on Coursera.  
The certificate program guided me from the foundations of web development (HTML, CSS, JavaScript) to advanced front-end frameworks like **React** and **UI/UX design principles**.  

Each folder represents a course from the certificate program and contains final projects.

## Project Highlights

### Introduction to Front-End Development
Created a **personal portfolio webpage** using HTML and CSS, focusing on layout, structure, and accessibility.

### Programming with JavaScript
Built small JS projects and exercises using functions, loops, objects, and event handling.

### HTML and CSS in Depth
Developed **responsive forms** and layouts using Flexbox and CSS Grid, with improved semantics and accessibility.

### React Basics & Advanced React
Implemented **interactive React apps** using components, state management, props, and hooks.

### Principles of UI & UX Design
Designed a **user-centered interface prototype** following core UI/UX principles.  Applied the **design thinking process** to create wireframes, prototypes, and interactive layouts using **Figma**.

### Capstone Project: Little Lemon
A full-scale **restaurant reservation and booking website** built in React, showcasing advanced component structure, responsive UI, and data fetching.
